R Libraries

Two R-libraries are used:

Function to convert LLR to ILD for the ICI = 0 condition

llr_to_ild <- function(llr) {
  lmax <- 20 * log10 (10^(0/20) + 10^(llr/20))
  lmin <- 20 * log10 (10^(0/20) + 10^((llr-10)/20))
  lmax-lmin
}

Data for Figure 2, threshold trend plot for each stimulus condition

Thresholds were caluclted using Hamiltonian Monte Carlo estimation of three parametets, \(\alpha, \beta, \lambda\), of this function:


\(\psi(x; \alpha, \beta, \lambda) = 0.5 + (0.5 - \lambda)\Phi(\alpha + x\beta)\),

where \(\psi\) is the proportion of correct answers, \(x\) is the lag-lead ratio, LLR, in dB, \(\Phi\)ï is the normal cumulative distribution function with location parameter \(\alpha\) and scale parameter \(\beta\), and \(\lambda\) is the lapse-rate parameter (cf. Kingdom & Prins, 2016, Ch. 4). A threshold was calculated by inverting the functiont to the find the LLR corresponding to 75 % correct responses:


\(\psi^{-1}(0.75) = (\Phi^{-1}(\frac{0.25}{0.5 - \lambda}) - \alpha) / \beta\),

STAN model

The STAN model was saved in a separate file called psychomteric_function.stan. Here is the content of the file:

data {
int<lower=1> N;
vector[N] x;
int<lower=0,upper=1> y[N];
}
parameters {
real alpha;
real<lower=0> beta;
real<lower=0> lambda;
}
model {
alpha ~ cauchy(0,20);
beta ~ cauchy(0,2);
lambda ~ cauchy(0,0.02);
for (i in 1:N)
{
y[i] ~ bernoulli(0.5 + (0.5-lambda)*Phi(alpha + beta * x[i]));
}
}



Function that calls stan() and outputs threshold estimates.

Input:

  • ici — Value of inter-click interval (called tau in the raw data files), numeric
  • startday — First day of 10 day period for which threshold is calculated, numeric
  • chains — Number of chains of sampling from the posterior, numeric
  • iter — Number of samples drawn, numeric
  • warmup – Number of warmup samples, hasd to be less than iter, numeric

Output: Vector with estimates calculated from the posterior distribution, and diagnostics from STAN (n_eff = effective samples; Rhat = convergence, should be 1.00). Bold entries were used in the manuscript to draw Figure 2.

  • startday — Same as input
  • thmean — Mean threshold
  • thmedian — Median threshold
  • th95lo — Lower 95 % credibility interval of threshold
  • th95hi — Upper 95 % credibility interval of threshold
  • meanlapse — Mean of lapse rate parameter
  • medianlapse — Median of lapse rate parameter
  • maxlapse — Maximum of lapse rate parameter
  • a_eff — Diagnostics for \(\alpha\)
  • b_eff — Diagnostics for \(\beta\)
  • lambda_eff — Diagnostics for \(\lambda\)
  • a_Rhat — Diagnostics for \(\alpha\)
  • b_Rhat — Diagnostics for \(\beta\)
  • lambda_Rhat – Diagnostics for \(\lambda\)

Global variable required: data.frame dd, a cleaned version of the raw data file (STAN doesn’t like files with unused variables), created below.

dd <- d[, c(4, 9, 13)]  # Only d$day, d$ratio and d$pc
moving <- function(ici, startday = 1, chains = 1, iter = 1e4, warmup = 2e3) {
  
  # Compile data
  g <- dd[dd$tau == ici & dd$day >= startday &  dd$day < (startday + 10),]
  y <- as.integer(g$pc)  # Answer
  x <- g$ratio  # Level
  N <- as.integer(length(y))
  
  # Run STAN
  model <- stan("psychometric_function.stan", data = list(N = N, x = x, y = y), 
              chains = chains, iter = iter, warmup = warmup)
  print(model)  #Print output
  
  # Get sampled parameters from the Posterior distribution
  a <- model@sim$samples[[1]]$alpha[(warmup + 1):iter]
  b <- model@sim$samples[[1]]$beta[(warmup + 1):iter]
  cc  <- model@sim$samples[[1]]$lambda[(warmup + 1):iter]
  
  # Get diagnostics
  n_eff <- summary(model, pars = c("alpha", "beta", 'lambda'), 
                   probs = c(0.1, 0.9))$summary[, 6]
  R_hat <- summary(model, pars = c("alpha", "beta", 'lambda'), 
                   probs = c(0.1, 0.9))$summary[, 7]
  
  # Use sampled parameters to calculate threshold estimate, by inverting function at 75 % correct
  thpost <- (qnorm((0.75-0.5)/(0.5 - cc)) - a) /  b  # Threholds
  ci <- quantile(thpost, prob = c(0.025, 0.975), na.rm = TRUE)  # 95 % interval
  estimate <- mean(thpost, na.rm = TRUE)  # Mean threshold
  mdn <- median(thpost, na.rm = TRUE)
  
  # Lapse rate estimates from parameter lambda
  lapse <- c(mean(cc, na.rm = TRUE), median(cc, na.rm = TRUE), max(cc, na.rm = TRUE))
  
  # Compile output
  out <- c(startday, estimate, mdn, ci, lapse, n_eff, R_hat)
  names(out) <- c('startday', 'thmean', 'thmedian', 'th95lo', 'th95hi', 
                  'meanlapse', 'mdnlapse', 'maxlapse',
                  'a_eff', 'b_eff', 'lambda_eff',
                  'a_Rhat', 'b_Rhat', 'lambda_Rhat')
  out
}

STAN model for comparing first and last 10 days

To compare thresholds from days 1-10 with days 51-60, parameters of this function was estimated:


\(\psi(x; \alpha, \beta, \gamma, \eta, \lambda) = 0.5 + (0.5 - \lambda)\Phi(\alpha + \beta x + \gamma D + \eta Dx)\),


where \(\psi\) is the proportion of correct answers, \(x\) is the lag-lead ratio, LLR in dB, \(D\) is a dummy variable coded 0 for days 1-10 and 1 for days 51-60, \(\Phi\) is the normal cumulative distribution function with location parameters \(\alpha\), \(\gamma\) and scale parameters \(\beta\), \(\eta\), and \(\lambda\) is the lapse-rate parameter. The function for days 1-10 thus have location parameter \(\alpha\) and scale parameter \(beta\) (because \(D = 0\)). The function for days 51-60 have location parameter \(=\alpha + \gamma\) and scale parameter \(=beta + \eta\) (because \(D = 1\)). A threshold was calculated by inverting the functiont to the find the LLR corresponding to 75 % correct responses:


The STAN model was saved in a separate file called prepost.stan. Here is the content of the file:

data {
int<lower=1> N;
vector[N] x;
int<lower=0,upper=1> y[N];
int<lower=0,upper=1> z[N];
}
parameters {
real alpha;
real<lower=0> beta;
real gamma;
real eta;
real<lower=0> lambda;
}
model {
alpha ~ cauchy(0,20);
beta ~ cauchy(0,2);
gamma ~ cauchy(0,3);
eta ~ cauchy(0,3);
lambda ~ cauchy(0,0.02);
for (i in 1:N)
{
y[i] ~ bernoulli(0.5 + (0.5-lambda)*Phi(alpha + beta * x[i] + gamma * z[i] + eta * x[i] * z[i]));
}
}

Global variable required: data.frame g, a cleaned version of the raw data file with an extra variable g$prepost, created below.

g <- d[(d$day < 11 |  d$day > 50),]  # Only Day 1-10 or Day 51-60
g$prepost <- (1 * g$day > 50)  # 0 if Day 1-10, 1 if Day 51-60


firstlast <- function(ici, chains = 1, iter = 1e4, warmup = 2e3) {
  # Compile data
  y <- as.integer(g$pc[g$tau == ici])
  x <- g$ratio[g$tau == ici]
  z <- as.integer(g$prepost[g$tau == ici])
  N <- as.integer(length(y))
  
  # Run STAN
  model <- stan("prepost.stan", data = list(N = N, x = x, y = y, z = z), 
              chains = chains, iter = iter, warmup = warmup)
  print(model)

  # Get sampled parameters from the Posterior distribution
  alpha <- model@sim$samples[[1]]$alpha[(warmup + 1):iter]
  beta <- model@sim$samples[[1]]$beta[(warmup + 1):iter]
  gamma <- model@sim$samples[[1]]$gamma[(warmup + 1):iter]
  eta <- model@sim$samples[[1]]$eta[(warmup + 1):iter]
  lambda <- model@sim$samples[[1]]$lambda[(warmup + 1):iter]
  
  # Get diagnostics
  n_eff <- summary(model, pars = c("alpha", "beta", 'gamma', 'eta', 'lambda'), 
                   probs = c(0.1, 0.9))$summary[, 6]
  R_hat <- summary(model, pars = c("alpha", "beta", 'gamma', 'eta', 'lambda'), 
                   probs = c(0.1, 0.9))$summary[, 7]
  
  # Use sampled parameters to calculate threshold estimate, by inverting function at 75 % correct,
  # separately for first 20 days (pre) and last 10 days (post), and for their difference
  thpre <- (qnorm((0.75-0.5)/(0.5 - lambda)) - alpha) /  beta
  ci_pre <- quantile(thpre, prob = c(0.025, 0.975), na.rm = TRUE)
  thpost <- (qnorm((0.75-0.5)/(0.5 - lambda)) - (alpha + gamma)) / (beta + eta)
  ci_post <- quantile(thpost, prob = c(0.025, 0.975), na.rm = TRUE)
  thdiff <- thpost - thpre
  ci_diff <- quantile(thdiff, prob = c(0.025, 0.975), na.rm = TRUE)
  estimates <- c(mean(thpre, na.rm = TRUE), mean(thpost, na.rm = TRUE), 
                 mean(thdiff, na.rm = TRUE))
  mdns <- c(median(thpre, na.rm = TRUE), median(thpost, na.rm = TRUE),
          median(thdiff, na.rm = TRUE))
  
  # Lapse rate estimates from parameter lambda
  lapse <- c(mean(lambda, na.rm = TRUE), median(lambda, na.rm = TRUE), max(lambda, na.rm = TRUE))
  
  # Compile output
  out <- c(ici, estimates, mdns, ci_pre, ci_post, ci_diff, lapse, n_eff, R_hat)
  names(out) <- c('ici', 'thmean_pre', 'thmean_post', 'thmean_diff',
                  'thmedian_pre', 'thmedian_post', 'thmedian_diff',
                  'th95lo_pre', 'th95hi_pre', 'th95lo_post', 'th95hi_post', 
                  'th95lo_diff', 'th95hi_diff', 
                  'meanlapse', 'mdnlapse', 'maxlapse',
                  'alpha_eff', 'beta_eff', 'gamma_eff', 'eta_eff', 'lambda_eff',
                  'alpha_Rhat', 'beta_Rhat', 'gamma_Rhat', 'eta_Rhat', 'lambda_Rhat')
  out
}
